# Trenches Survivor Card 2025

Official generator for the 2025 Trenches Survivor Pass.

## Features

- **Identity Verification**: Input your name and upload your profile picture.
- **PnL Tracking**: Display your 2025 Profit/Loss (verified rekt or moon).
- **Official Design**: High-quality, holographic-style card design.
- **Instant Minting**: Download your card as a PNG image.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the URL shown (usually http://localhost:5173).

## Tech Stack

- React
- Vite
- html2canvas
